## Build & Run Commands

### Desktop
```bash
cd desktop/FingerprintAttendanceApp
dotnet restore
dotnet build --configuration Release
dotnet run
```

### Backend
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
